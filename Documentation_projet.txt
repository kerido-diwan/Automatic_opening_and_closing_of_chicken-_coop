Projet fermeture et ouverture crépusculaire automatique d'un poulailler.

	Ici le but est de créer un système pour ouvrir ou fermer une trappe installer dans un poulailler. Nous devons donc coder (ici) un carte Arduino.
	Le but est que la trappe se ferme quand il fait nuit et s'ouvre quand il fait jour.


Matériel:

	Comme matériel utilisé nous avons:
	Arduino UNO.
	Carte L298N (pour contrôler le moteur).
	Une photorésistance.
	Bouton poussoir.
	Résistance 220 ohm.
	Un moteur 12V.
	Une alimentation 12V.
	Câbles.

	Apres ça c'est le matériel pour le système de trappe poulie et tout.
	C'est vous qui voyez.


Code:
	Dans mon code le moteur est contrôlé par les broches in1 et in2 (carte L298N). Par endroit in1 est activé mais pas in2, inversement aussi, ou même 0/1 	pour les deux. Cela est le cas car ça donne le sens de rotation pour le moteur. Qui dépend aussi de votre code et branchement. Quand un des deux in 	est égale a 1 et l'autre 0 le moteur tourne dans un sens ou l'autre. Si in1 et in2 sont égale à 0 ou 1 le moteur est a l'arrêt.
